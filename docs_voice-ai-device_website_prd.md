## 一、项目概述

### 1.1 项目背景

基于 [docs_voice-ai-device_requirements.md](../docs_voice-ai-device_requirements.md) 的产品定位，建设官网首页中英文双版本，用于产品展示和商务接洽。

### 1.2 目标

- 建立公开展示渠道，让用户/投资人/合作伙伴了解产品
- 收集潜在客户咨询意向
- 支持中英文双语访问

### 1.3 技术选型

- **CMS框架**：Halo（https://github.com/halo-dev/halo）
- **网站图片**：全部使用AI生成，不使用互联网图片

## 三、视觉风格

### 3.1 设计方向

- **风格**：温暖、专业、可信赖（养老+科技定位）
- **色调**：
  - 主色：暖橙色/琥珀色（温暖感）
  - 辅助色：深蓝/藏青（专业、可信）
  - 背景色：白色/浅灰
- **字体**：思源黑体/阿里巴巴普惠体（中文）、Inter（英文）
- **图片**：全部AI生成，风格统一温馨，避免使用真实老人照片

### 3.2 响应式断点

| 设备 | 宽度 |
|------|------|
| 移动端 | < 768px |
| 平板 | 768px - 1024px |
| 桌面 | > 1024px |

## 四、禁止出现的内容

| 禁止项 | 说明 |
|--------|------|
| 官方商城/在线购买/价格表 | 个人无执照，易被审核拒绝 |
| 无关内容（游戏、资讯等） | 专注养老AI产品 |
| 互联网图片 | 全部使用AI生成图片 |
| 免费二级域名 | 必须使用独立域名（.top/.com等） |
| 技术细节（如Picovoice） | 官网仅展示产品定位，不体现技术实现 |
| 空白页/占位页 | 至少填满5块内容 |
