## 四、语音交互

### 4.1 语音唤醒 + 对话流程

```
日常待机状态（低功耗唤醒监听）
    ↑
    │ 用户连续呼唤AI名字两遍（如"小智小智"）
    │
    ↓ 唤醒成功
AI立即回复"我在"
    ↓
进入对话状态（唤醒监听暂停）
    ↓
用户说话 → faster-whisper(base) → LLM → AI回复 → TTS
    ↓
对话结束（如10秒无活动）→ 返回待机状态（唤醒监听恢复）
```

**唤醒与对话模式互斥**：
- 唤醒模式下：openwakeword + WebRTC VAD 低功耗监听唤醒词
- 对话模式下：停止唤醒监听，使用 Silero VAD 持续录音检测语音
- 对话结束（10秒静音）后：自动恢复唤醒监听

**完整语音交互链路**：

```
唤醒词检测（openwakeword + WebRTC VAD前置过滤）
    ↓
语音活动检测（Silero VAD）← 对话模式使用
    ↓
语音转文字（faster-whisper base）← 说话内容识别，支持繁简转换
    ↓
大语言模型思考回复（Qwen3-4B + llama.cpp）← AI思考和生成回复
    ↓
文字转语音（Piper TTS zh_CN-huayan-medium）+ 屏幕显示 ← LLM流式输出文字，UI实时显示；TTS后台线程合成，语音略晚于文字（准同步）
```

### 4.2 唤醒词检测

| 参数 | 值 | 说明 |
|------|-----|------|
| 唤醒词 | AI名字 × 2 | 如"小智小智"、"小爱小爱" |
| 检测方式 | openwakeword + WebRTC VAD 前置过滤 | 实时监听，VAD过滤白噪声后送入openwakeword检测 |
| VAD模式 | 模式1 |  менее агрессивный，更适合中文唤醒词检测 |
| 响应延迟 | < 500ms | 唤醒后立即回复"我在" |
| 双触发窗口 | 1.5秒 | 两次检测需在1.5秒内完成 |
| 最小触发间隔 | 0.8秒 | 防止同一句话被识别为两次 |
| 待机模式 | 屏幕熄灭，AI不主动响应 | 降低功耗；唤醒词、触屏、按钮操作均可点亮屏幕 |

**唤醒检测流程**：
1. WebRTC VAD（模式1）检测语音帧，过滤白噪声和安静环境
2. 缓冲约300ms音频后送入openwakeword
3. openwakeword计算唤醒词置信度分数
4. 双触发检测：两次检测间隔需>0.8秒（确保是两次独立发音）
5. 唤醒触发后5秒冷却，防止重复触发

**openwakeword配置**：
- 支持中文唤醒词模型（xiao_ai, xiao_bu, xiao_zhi）
- CPU占用低，适合边缘设备
- 模型存储：`models/openwakeword/`
- 唤醒阈值：0.0005（适配VAD过滤后的音频）

**be-more-agent 优化（voice-ai-device-raspberry-pi 分支）**：
- 音量预筛选：唤醒前先检查音量是否 > 200，省约60% CPU开销
- 最近邻重采样：代替 scipy.signal.resample，降低 CPU 占用
- 采样率自动探测：自动检测麦克风支持的采样率（48k→44.1k→32k→16k）

```python
# 音量预筛选 - 来自 be-more-agent
audio_int16 = np.frombuffer(frame, dtype=np.int16)
if np.abs(audio_int16).mean() < VOLUME_THRESHOLD:  # 200
    continue  # 跳过唤醒词检测
```

**唤醒与对话模式切换**：
```
唤醒成功 → 停止唤醒监听 → 启动对话模式 → 10秒静音超时 → 停止对话模式 → 恢复唤醒监听
```

**对话模式**：
- 唤醒后进入持续对话状态
- Silero VAD检测语音，自动录音
- 说话后发送录音至STT转文字
- 10秒静音自动退出对话模式
- 对话结束后自动恢复唤醒监听

**be-more-agent 增强功能（voice-ai-device-raspberry-pi 分支）**：

| 功能 | 描述 | 代码位置 |
|------|------|----------|
| 双级对话记忆 | permanent_memory（持久化）+ session_memory（本次会话，10条滚动） | `app/memory.py` |
| Action 路由 | LLM 输出 JSON 结构化动作，如 `{"action": "search_web", "query": "..."}` | `app/action_router.py` |
| 流式 TTS | Piper raw PCM → sounddevice 直接播放，无文件 I/O | `app/stream_tts.py` |
| 视觉能力 | Moondream 图像描述 + rpicam-still 摄像头 | `app/vision.py`, `app/camera_pi.py` |
