## 七、运动轨迹记录

### 7.1 数据结构

```python
# 滚动缓冲区：最近5分钟（300条）
MAX_TRAJECTORY_BUFFER = 300

{
    "timestamp": 1745337580.123,  # Unix时间戳
    "pose": "standing"            # 姿态状态
}
```

### 7.2 轨迹用途

- 跌倒事件回溯：跌倒发生前后各30秒的姿态序列
- 事件日志存储：跌倒事件包含 `trajectory` 字段
- 查询API：`GET /pose/trajectory?seconds=30`

## 八、用药提醒

### 8.1 创建方式

| 方式 | 操作 |
|------|------|
| 语音创建 | 对AI说"提醒我每天8点吃降压药" |
| UI创建 | 用药页 → 输入药品名 + 时间 → 添加 |

### 8.2 触发流程

```
定时任务触发（APScheduler）
    ↓
TTS语音播报："王大爷您好，您该吃降压药了。"
    ↓
记录日志（triggered / confirmed / skipped）
```

### 8.3 删除方式

| 方式 | 操作 |
|------|------|
| 语音删除 | 对AI说"删除降压药的提醒"、"取消感冒药的提醒"等 |
| UI删除 | 用药页 → 点击提醒右侧【-】按钮 |
