# PRD 模板

> 三人协作格式：Robin 讨论 + 小精灵起草 + Claude Code 执行

---

```markdown
# [产品名] PRD

## Context（背景）
**讨论结论**：[我们聊的框架]

## Decisions（已决事项）
- D1: [明确决策]
- D2: [明确决策]

## Specs（功能规格）
### 功能 A
**目标**: ...
**验收标准**:
1. ...
2. ...

### 功能 B
...

## Open Questions（待讨论）
- Q1: [还没解决的]
- Q2: [需要跟 Robin 确认的]

## AI Implementation Notes（Claude Code 填写）
- 实现方案: ...
- 代码位置: ...
- 已知限制: ...

## Implementation Log（迭代记录）
- YYYY-MM-DD [角色]: 描述
```

---

## 使用说明

1. **新建 PRD**：复制这份模板到 `prd/[产品名].md`
2. **讨论流程**：Robin 和小精灵讨论 → 小精灵更新本文档
3. **执行移交**：Robin 启动 Claude Code → 基于本文档执行
4. **文档格式**：Context → Decisions → Specs → Open Questions，闭环
